## Vision

The long-term goal is to evolve static articles into **visual explanations** — animated diagrams (Manim), interactive schemas (D3.js), and eventually full video episodes. Text stays rigorous; visuals make it stick.

```
Article (Markdown + KaTeX)
    → Animated scenes (Manim / Motion Canvas)
    → Interactive exploration (D3.js / HTML)
    → Video episode (ffmpeg assembly + TTS narration)
    → Social media distribution (YouTube, LinkedIn, X)
```
